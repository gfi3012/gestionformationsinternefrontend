Présentation du projet

  • Présentation du sujet

    • Le projet « Mise en place d’une application de gestion des formations interne » consiste à mettre en place une application web pour la gestion et le suivi des cursus de formation interne pour les collaborateurs d’une société informatique. Elle permet, en outre, de définir les objectifs de la formation, détailler la démarche de formation et effectuer le bilan pour chaque formation.

  • Architecture applicative

    Angular, PrimeNG, Bootstrap
    Couche de services :
    Spring MVC
    Couche métier :
    Spring Boot
    Couche d'accès aux données :
    Spring Data JPA
    Couche de base de données :
    JPA, MySQL
